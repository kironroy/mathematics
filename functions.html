Higher-Order Functions

A higher-order function is a function that either:

  • Takes another function as a parameter
  • Returns a new function as its result

This enables patterns like reusability, abstraction, and modular composition, making your code more expressive and powerful.

mapAndSum function


function mapAndSum(fn) {
  return function(numbers) {
    return numbers.map(fn).reduce(function(total, val) {
      return total + val;
    }, 0);
  };
}

mapAndSum is a higher-order function because it accepts a function (fn) as input to control the logic of transforming array elements. Then it maps and reduces the data to return a total.
